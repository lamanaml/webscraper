# Web Scraper

## Overview
Web app that scrapes Vegetarian Times vegan recipes and displays the titles.  Also allows you to add notes.


### Live Site
https://stark-cliffs-46952.herokuapp.com/

### Github Repo

* Express
* Express-Handlebars
* Mongoose
* MVC
* Request (instead of Axios)
* Cheerio
